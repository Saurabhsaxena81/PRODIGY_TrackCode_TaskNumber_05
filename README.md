
## How to Run

1. Clone this repository to your local machine using `git clone`.
2. Open the `index.html` file in your web browser.

## Usage

1. Enter the name of a city or location in the input field.
2. Press the "Get Weather" button to fetch the current weather details for that location.
3. The application will display the temperature, weather description, humidity, wind speed, and more.
